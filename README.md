# OAS Bay - Auto Service Dashboard

A simple web dashboard for **Oyera Auto Service Bay Ltd**, an auto service station based in Uganda. This system helps the business move from manual record-keeping to a digital solution for tracking car services, parts, and billing.

## About

This project was built as a **Dashboard Assignment** for the Introduction to Software Engineering course at **Refactory** (CSE - June Cohort 2026).

The dashboard displays service records for cars that come in for engine oil changes, brake fluids, gearbox oil, brake pads, greasing, wheel alignment, and balancing. All prices are displayed in Ugandan Shillings (UGX).

## Tech Stack

- HTML
- CSS
- Bootstrap 5
